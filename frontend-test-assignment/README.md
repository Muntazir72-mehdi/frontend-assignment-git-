# Frontend Test Assignment

This project recreates a UI component from a Figma design using HTML, CSS, and JavaScript. The component is a card with page selection checkboxes and a "Done" button.

## Features

- Pixel-perfect recreation of the design
- Interactive checkboxes with "All pages" toggle functionality
- Clean, modern UI with shadows and rounded corners
- Responsive design
- No external dependencies

## Files

- `index.html`: The main HTML structure
- `styles.css`: CSS for styling the component
- `script.js`: JavaScript for interactivity

## How to Run

1. Open `index.html` in a web browser
2. The component will be displayed and fully functional

## Deployment

To deploy this project:

1. Upload the files to a GitHub repository
2. Enable GitHub Pages in the repository settings
3. The live version will be available at `https://yourusername.github.io/repository-name`

Alternatively, you can use Netlify, Vercel, or CodePen for deployment.

## Interactivity

- Check/uncheck individual page checkboxes
- Use the "All pages" checkbox to select/deselect all pages at once
- Hover over the "Done" button for visual feedback
